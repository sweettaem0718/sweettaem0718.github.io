# TAEMIN4CUT

TAEMIN4CUT(태민네컷)은 웹캠으로 4컷 사진을 촬영하고, 선택한 프레임과 합성해 최종 이미지를 저장하는 정적 웹 프로젝트입니다.

## 프로젝트 분석 요약

- 형태: HTML/CSS/JavaScript 기반 정적 웹앱
- 핵심 기능:
  - 프레임 선택 후 4컷 촬영
  - 실시간 프레임 합성 미리보기(canvas)
  - 카메라 미러 ON/OFF
  - 최종 이미지 JPG 다운로드
  - 다국어 지원(한국어/영어/일본어/중국어)
  - 동의 모달, 모바일 최적화 안내, 카메라 권한 안내
- 데이터/자산 구조:
  - `frames.json`에서 프레임 메타데이터(크기, 슬롯, 이미지 경로) 로딩
  - `images/frames/*`의 베이스/오버레이 이미지로 합성
- 개인정보 처리 방식:
  - 코드 기준, 촬영 이미지는 서버 업로드 없이 브라우저 내 처리 후 로컬 다운로드

## 디렉터리 구조

```text
.
├─ index.html
├─ frames.json
├─ css/
│  └─ styles.css
├─ js/
│  └─ app.js
└─ images/
   └─ frames/
```

## 실행 방법

정적 파일 프로젝트이므로 아래 중 하나로 실행할 수 있습니다.

1. 로컬에서 `index.html` 직접 열기
2. 정적 서버 실행 후 접속 (예: `python -m http.server`)
3. GitHub Pages 등 정적 호스팅에 배포

## 이용 및 권리 안내

- 본 프로젝트는 현재 상업적 사용 및 수익창출을 하고 있지 않습니다.
- 본 프로젝트 내 모든 리소스는 제작자 본인이 수정한 결과물이며, 2차 사용은 불가능합니다.
- 합성 이미지는 [포토이즘](https://x.com/photoism_kr/status/1824340315718685073) 내 사진 기반으로 재가공되었습니다.
- 원작자의 요청이 있을 경우, 본 프로젝트의 배포는 중단될 수 있습니다.
